Please use this tool responsibly. The images you download are not public domain. If you do not have permission to reupload, do not reupload. This is for your personal archive only.

It's slow. Keep rooting it on while it works.

Files are saved [submission number]_[original file name]. Submissions marked explicit are saved in their own folder.

Run "python3 main.py [usernumber]" to download your favorite dad's gallery. Their usernumber is in the URL of their profile

The script will automatically exit if a previously downloaded image is found. Use the -f option (ex: "python3 main.py 13688 -f"), the script will keep moving past any already downloaded files without redownloading them. Use this if something breaks during your last download session and the full gallery wasn't completed.
